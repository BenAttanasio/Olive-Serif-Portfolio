OLIVE SERIF PORTFOLIO
=====================

Personal portfolio site for Ben Attanasio.

Dark editorial aesthetic — olive/sage green accents, cream typography, Playfair Display serif headings, and clean dark charcoal. Single HTML file, no dependencies, no build step.

LIVE SITE
---------
https://benattanasio.com

STACK
-----
- Pure HTML / CSS / JavaScript
- Single file: index.html
- Google Fonts: Playfair Display (loaded via CDN)
- Zero npm, zero framework, zero build process

DESIGN SYSTEM
-------------
- Background: #131313 (neutral dark charcoal)
- Accent: #7a8c68 (olive/sage green)
- Text: #eae6dc (cream)
- Serif display font: Playfair Display
- Glassmorphism cards with subtle borders
- Mouse parallax on ambient background shapes
- Sticky ACS community banner at top
- Fully responsive (900px breakpoint)

SECTIONS
--------
1. Hero — Intro with community CTA above the fold
2. The Engine — Enterprise experience (Tricentis, MetLife)
3. Agentic Coding Society — Community deep-dive
4. Sift — AI research tracking tool (sift.scalingsidekick.com)
5. Footer — "Your move." with links
